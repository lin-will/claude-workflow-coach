# Claude Workflow Coach

A web app that guides non-engineers through professional Claude Code workflows — from dev environment setup to agentic prompting and MCP integration — so they can use AI as a true development partner instead of a basic code generator.

> **Status:** In active development. Currently at PRD / project setup stage.

## The Problem

Non-engineers struggle to use Claude Code as a real development tool to enhance their workflows because they're not experienced with professional development environments and the existing possibilities of Claude Code. They continue to use AI as a search engine and basic code generator instead of a comprehensive force multiplier.

## The Solution

The Claude Workflow Coach onboards users by role, tools, and technical level, then generates a personalized AI workflow setup guide. A scoped follow-up chat interface lets users drill into specific topics. The goal: every user leaves having adopted at least one concrete workflow improvement.
